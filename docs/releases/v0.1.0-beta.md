# Recap v0.1.0-beta

한국어 회의를 로컬에서 녹음하고, 원문 근거가 있는 Decision Wiki로 정리하는 Apple Silicon용 오픈소스 도구입니다.

이번 릴리즈는 "회의 전사 앱"보다 한 단계 좁고 분명한 방향을 잡는 데 초점을 둡니다. 회의가 끝난 뒤 긴 전사문만 남기는 것이 아니라, 결정사항과 액션아이템을 Markdown Wiki 레이어로 정리하고 timestamp 근거와 함께 다시 찾을 수 있게 만드는 것이 핵심입니다.

## Highlights

- Apple Silicon Mac에서 로컬로 실행되는 한국어 회의 녹음/전사 파이프라인
- mlx-whisper 기반 STT, pyannote 화자 분리, 로컬 LLM 교정/요약
- 회의 원문 검색을 위한 ChromaDB + SQLite FTS5 RAG 인덱스
- 결정사항과 액션아이템을 원문 timestamp 근거와 함께 정리하는 Decision Wiki 흐름
- Wiki 전용 BM25/FTS5 + e5-small 하이브리드 검색
- macOS 메뉴바 앱과 웹 SPA UI
- STT/LLM 모델 선택을 위한 설정 화면

## Install

```bash
git clone https://github.com/notadev-iamaura/meeting-transcriber.git
cd meeting-transcriber
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
bash scripts/install.sh
```

화자 분리를 사용하려면 HuggingFace에서 pyannote gated model 접근 동의와 read token 설정이 필요합니다. 자세한 단계는 README의 빠른 시작 섹션을 확인하세요.

## Known Limits

- Apple Silicon Mac 전용입니다. Intel Mac, Linux, Windows는 현재 지원하지 않습니다.
- Python 3.11 또는 3.12를 권장합니다. Python 3.13 이상은 ChromaDB 네이티브 바인딩 이슈가 있을 수 있습니다.
- Decision Wiki 자동 생성 결과는 원문 timestamp 근거와 함께 확인하는 흐름을 전제로 합니다.
- 화자 분리 모델 최초 다운로드에는 HuggingFace 계정과 gated model 약관 동의가 필요합니다.
