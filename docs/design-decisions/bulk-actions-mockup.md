### 3.2 selection mode 활성 — 컨텍스트 액션 바 적응

```
┌──────────────────────────────────┐
│ ┌──────────────────────────────┐ │
│ │ BULK ACTION BAR (44px sticky)│ │
│ │                              │ │
│ │ 👤 [2]                  [✕]  │ │ ← 1행: 카운트 (사람 아이콘+숫자) + 닫기
│ │  ↑ aria-label="2개 선택됨"   │ │   "해제" 텍스트와 ⌘Esc <kbd> 모두 숨김
│ │                              │ │   aria-label="선택 해제" 보존
│ │ [✎] [≡] [✎+≡]                │ │ ← 2행: 액션 (아이콘만, 라벨 텍스트 숨김)
│ │  ↑ ↑   ↑ aria-label 풀텍스트:│ │   ✎=전사 / ≡=요약 / ✎+≡=전사+요약
│ │       "전사+요약"            │ │   long-press / hover 시 네이티브 툴팁
│ │                              │ │
│ │ ─────────────────────────── │ │
│ │  // 모바일: 2-row stack      │ │
│ │  // height: auto (44 → 88px) │ │
│ │  // padding: 8px 12px        │ │
│ └──────────────────────────────┘ │
├──────────────────────────────────┤
│ Sidebar (콘텐츠 영역과 stack)      │
│                                  │
│  ☑ ● 회의 A  selected            │
│  ☑ ● 회의 B  selected            │
│  ☐ ● 회의 C                     │
│                                  │
└──────────────────────────────────┘
```

**모바일 적응 규칙** (review-1b 정책 §2 ~ §3 반영):
- **컨텍스트 액션 바 레이아웃**: `flex-direction: column` (또는 `flex-wrap: wrap`) 으로 2-row stack
- **액션 3-버튼**: `flex: 1` 로 균등 분할 (각 ~33%, 모바일 375px 기준 약 110px)
  - **라벨 텍스트 숨김**: 아이콘만 표시 (e.g. ✎ / ≡ / ✎+≡ 또는 SVG)
  - **`aria-label` 로 텍스트 보존**: e.g. `aria-label="전사+요약"` — 스크린리더와 long-press/hover 툴팁에 노출
  - **글자 잘림 방지**: 라벨이 아예 없어 잘림 위험 0. 기존 `min-width: 0` + `text-overflow: ellipsis` 백업 안전장치
- **카운트 라벨 축약**: "3개 선택됨" → 사람 아이콘 (👤 또는 SVG) + "3" 숫자만 표시. `aria-label="3개 선택됨"` 로 스크린리더 보존
- **`[✕] 해제 ⌘Esc` 모바일 처리**: "해제" 텍스트 + `<kbd>Esc</kbd>` 둘 다 숨김. ✕ 아이콘만 표시 + `aria-label="선택 해제"`
  - 미디어 쿼리: `@media (max-width: 640px) { .bulk-action-bar__dismiss .label, .bulk-action-bar__dismiss kbd { display: none; } }`
- 사이드바 ↔ 콘텐츠 stack 모드 (기존 SPA 모바일 레이아웃 유지)
- 드롭다운 메뉴는 모바일에서 `min-width: calc(100vw - 32px)`, `max-width: 320px` 로 적응

### 3.3 모바일 hover 부재 처리

`hover` 가 모바일에서 동작하지 않으므로:
- 체크박스: 항상 `opacity: 1` (모바일 selection mode 명시적 진입 방식)
- 또는: `@media (hover: none) { .meeting-item-checkbox { opacity: 1 } }` 미디어 쿼리로 분기
- 시각 잡음을 줄이기 위해 모바일에서는 체크박스 색을 `var(--text-muted)` 보더로만 (채움 없음)
- **툴팁 대안**: 라벨 숨김 환경에서 사용자가 텍스트 의미를 확인하려면 long-press 시 네이티브 `title` 또는 `aria-label` 기반 툴팁이 표시됨 (모바일 OS 기본 동작)

### 키보드 시퀀스 (마우스 없이)

```
[t=0s]  Tab → 사이드바 진입 (첫 항목 focus)
[t=0.5s] ↓↓ → 회의 C 로 이동 (focus ring 표시)
[t=1.0s] Space → 회의 C 토글 → selection mode 진입, 액션 바 등장

[t=2.0s] Space → 회의 D 토글
```
